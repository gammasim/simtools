Generalize filling and usage of histogram based on triggered events with several major improvements:

At a generic application to fill and write triggered-event histograms from reduced event-data files
(`simtools-write-trigger-histograms`). The following application are modified to use this generic histograms:

- `simtools-compare-productions`
- `simtools-production-derive-corsika-limits`
- `simtools-plot-simulated-event-distribution`
- `simtools-production-derive-monte-carlo-statistics`

Improve the estimation of required Monte Carlo statistics in `simtools-production-derive-monte-carlo-statistics`
by using the generic histogram and by providing a more robust estimation of the relative statistical uncertainty.

Improve file-type related utilities:

- add a utility application `simtools-inspect-file` to print out main structure and metadata of a simtools-generated file.
- add new modules `simtools.io.file_type` and  `simtools.io.inspect_file` with helper functions to inspect and identify files.
