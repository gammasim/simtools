Changes to the setting of seeds for sim_telarray:

- `sim_telarray_random_instrument_instances` and `sim_telarray_instrument_seeds` to generate a random seed file for the given number of configurations.
- new parameter `set_seed_for_random_instrument_instances` to set both instrument (configuration) and simulation seeds (e.g., --set_seed_for_random_instrument_instances 1745,290). This parameter completely fixes the seeds and should only be used for testing purposes.
