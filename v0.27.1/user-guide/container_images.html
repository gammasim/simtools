Container Images#

OCI-compatible container images are available for simtools users and support both application and development use cases. Runtimes such as Docker, Podman, or Apptainer can be used.

Pre-built Images#

Below is an overview of the most-relevant pre-built container images available from the simtools package registry.

All images use almalinux:9.5 or almalinux:9.5-minimal with Python 3.12 as base images.
